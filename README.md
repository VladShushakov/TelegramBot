# YouTube Mini App для Telegram

Telegram Mini App для просмотра YouTube видео без VPN в России. Использует альтернативные фронтенды (Invidious, Piped) для обхода блокировок.

## Возможности

- Поиск видео по YouTube без VPN
- Воспроизведение видео через альтернативные фронтенды
- Поддержка прямых ссылок на YouTube
- Три источника для выбора (Invidious, Piped, YouTube)
- Адаптивный дизайн под мобильные устройства
- Темная тема, интеграция с Telegram WebApp API

## Как это работает

Приложение использует публичные инстансы альтернативных фронтендов YouTube:

- **Invidious** - открытый альтернативный фронтенд YouTube
- **Piped** - приватная альтернатива YouTube с современным дизайном

Эти сервисы работают как прокси, позволяя просматривать контент YouTube без прямого подключения к его серверам.

## Установка и настройка

### 1. Получение Telegram Bot Token

1. Найдите [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте команду `/newbot`
3. Следуйте инструкциям для создания бота
4. Сохраните полученный токен

### 2. Настройка хостинга

Вы можете разместить это приложение на любом бесплатном хостинге с поддержкой HTTPS:

#### Варианты хостинга:

- **GitHub Pages** (бесплатно)
- **Vercel** (бесплатно)
- **Netlify** (бесплатно)
- **Cloudflare Pages** (бесплатно)
- Любой свой сервер с HTTPS

#### Пример развертывания на GitHub Pages:

1. Создайте репозиторий на GitHub
2. Загрузите файл `index.html`
3. В настройках репозитория включите GitHub Pages (Settings > Pages)
4. Выберите ветку (обычно `main` или `master`)
5. Ваша ссылка будет: `https://ваш-username.github.io/название-репозитория/`

### 3. Создание Telegram Mini App

1. Перейдите к [@BotFather](https://t.me/BotFather)
2. Отправьте `/newapp`
3. Выберите вашего бота
4. Введите короткое имя (на латинице, 3-32 символа)
5. Введите название (на русском, 1-32 символа)
6. Введите описание (опционально)
7. Загрузите фото (опционально)
8. **Вставьте URL вашего приложения** (ссылка с шага 2)
9. Настройте содержимое Web App (опционально)

### 4. Тестирование

1. Откройте вашего бота в Telegram
2. Нажмите на меню бота или кнопку запуска
3. Mini App откроется в Telegram

## Использование

### Поиск видео
1. Введите название видео в поле поиска
2. Нажмите "Найти" или Enter
3. Выберите видео из результатов

### Прямая ссылка
Просто вставьте любую ссылку YouTube в поле поиска:
- `https://youtube.com/watch?v=VIDEO_ID`
- `https://youtu.be/VIDEO_ID`
- Или просто ID видео: `VIDEO_ID`

### Переключение источника
Если один источник не работает, переключитесь на другой:
- **Invidious** - основной рекомендуемый источник
- **Piped** - альтернативный источник
- **YouTube** - может работать с VPN

## Кастомизация

### Изменение источников

В файле `index.html` найдите объект `SOURCES` и измените URL инстансов:

```javascript
const SOURCES = {
    invidious: {
        search: (q) => `https://your-invidious-instance.com/api/v1/search?q=${encodeURIComponent(q)}`,
        embed: (id) => `https://your-invidious-instance.com/embed/${id}`,
        // ...
    },
    // ...
};
```

## Безопасность

- Приложение не собирает личные данные
- Использует публичные API альтернативных фронтендов
- Все запросы идут напрямую от браузера пользователя
- Не требует серверной части

## Ограничения

- Некоторые видео могут быть недоступны из-за ограничений прав владельца
- Публичные инстансы могут быть перегружены или недоступны
- Скорость загрузки зависит от выбранного инстанса

## Troubleshooting

### Видео не воспроизводится
- Попробуйте переключить источник (кнопки вверху)
- Проверьте, доступен ли инстанс в браузере
- Попробуйте другой публичный инстанс

### Ошибка подключения
- Убедитесь, что URL приложения правильный и доступен
- Проверьте, что используется HTTPS
- Попробуйте другой источник

### Mini App не открывается
- Убедитесь, что URL в настройках бота правильный
- Проверьте, что хостинг поддерживает HTTPS
- Очистите кэш Telegram и попробуйте снова
